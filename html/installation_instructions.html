Installation Instructions¶

Requirements¶

  • Python 2.x, 2.7 or superior
  • NumPy 1.6 or superior
  • VTK 5.x installed along with its python wrappings
  • nibabel 1.3.x

All of these can be easily obtained from pre-packaged distributions suck as Canopy or Anaconda. In these cases, the packages corresponding to VTK and nibabel will need to be added.

Installation¶

Once these requirements are met, the installation is as follows:

Downloading the source code from git:

git clone http://github.com/demianw/tract_querier.git

Installing:

cd tract_querier
python setup.py install

Now you can check if the installation worked:

tract_querier --help
